# 天机阁 TianjiOracle · 概念与部署

## 一、产品定位

在原版"小小东算命"基础上做的**增强版算命**，保留手相/面相的传统命理内核，但应用了一系列体验与变现升级。

**核心升级点（相比你的原版）**：

| 升级项 | 原版 | 天机阁增强版 |
|--------|------|-------------|
| 结果形态 | 一段分析文字 | 命格卡（称号+稀有度+证据链+批文+钩子） |
| 钩子机制 | 无 | 命格称号 + 稀有度攀比 + "未解关隘"悬念 |
| JSON稳定性 | 手动剥markdown，偶尔崩 | responseSchema 强制结构化，不崩 |
| 视觉风格 | 赛博科技蓝（有AI感） | 朱砂金·命理风（去AI感，更神秘） |
| 分享载体 | 普通海报 | 命格批文卡（带称号炫耀点） |
| 字体 | 黑体（软件感） | 宋体/衬线（古籍感） |

借用紫微斗数"十四主星"概念，预设10个命格（可扩展到14个）：

|---------|------|--------|
| 紫微帝星格 | 紫微 | ★★★★★ |
| 破军开拓格 | 破军 | ★★★★★ |

| 武曲财星格 | 武曲 | ★★★★☆ |
| 廉贞次桃花格 | 廉贞 | ★★★★☆ |
| 天府库藏格 | 天府 | ★★★★☆ |
| 太阳普照格 | 太阳 | ★★★☆☆ |

## 三、监管合规处理

虽然是算命主题，但做了软处理：
- ✅ 首页明确"娱乐声明"
- ✅ Prompt 中约束"不涉及健康/寿命/婚姻预测"
- ✅ 批文聚焦"性格特质/事业方向/行动建议"，弱化宿命论
- ⚠️ **注意**：算命主题在小程序/App上架仍有较高风险，纯H5+自有域名相对安全。如果要上小程序，建议用 soul-mirror（前世/平行宇宙）那一版。

## 六、三个版本对比（帮你决策）

| 维度 | xiaoxiaodong-fortune（原版） | tianji-oracle（增强算命） | soul-mirror（趣味身份） |
|------|---------|---------|---------|
| **主题** | 手相/面相算命 | 手相/面相算命 | 前世/平行宇宙身份 |
| **定位** | 工具型算命 | 命格卡·游戏化算命 | 趣味身份生成器 |
| **监管风险** | 高 | 中高 | 低 |
| **分享性** | 中 | 高（命格称号+稀有度） | 高（身份标签+稀有度） |
| **小程序可上架** | 难 | 难 | 较易 |
| **适合场景** | — | 纯H5推广，算命受众 | 全平台，年轻人裂变 |
| **视觉** | 赛博科技 | 朱砂金命理 | 墨金东方+赛博双主题 |
| **技术架构** | WIF+Gemini | WIF+Gemini+Schema | WIF+Gemini+Schema |

**我的建议**：
- 想做**算命受众、走H5私域**：用 tianji-oracle
- 想做**年轻人裂变、未来上小程序**：用 soul-mirror
- 两个可以**同时跑**，用不同域名测试哪个数据好，再all in
